# Triage label vocabulary

Default vocabulary — each role's label string equals its name. No overrides.

| Role | Label |
|---|---|
| Maintainer needs to evaluate | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Fully specified, AFK-ready | `ready-for-agent` |
| Needs human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

These labels don't exist in the GitHub repo yet (freshly created) — create them on first use (`gh label create`) rather than assuming they're present.
