# FreqTrace

A real-time audio analysis tool for live sound environments. Visualize sounds, track frequencies, and identify resonances and feedback on the fly.

## Features

- **Real-time Spectrograph** — Live frequency spectrum visualization
- **Frequency Tracker** — Monitor and track specific frequencies, resonances, and feedback
- **SPL Meter** — Simple sound pressure level metering
- **Signal Generator** — Generate test tones and signals

## Requirements

- macOS (primary platform; iOS support may come later)
- Xcode 16+
